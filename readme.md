# TodoList - Meteor.js

## Install Meteor.js
* Linux / OSX
```bash
curl https://install.meteor.com/ | sh
```

* Windows
```bash
choco install meteor
```

## Clone repo
```bash
git clone https://github.com/vishnuchandrappan/todolist-meteor.git
```

## Install dependencies
```bash
cd todolist-meteor
meteor npm install
```

## start dev server
```bash
meteor
```
